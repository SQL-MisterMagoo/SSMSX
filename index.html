SSMSX

Sql Server Management Studio Editor Extensions

Adding quick fixes

This project enables developers/dbas to create simple extensions for SSMS. The solution contains a VSIX project that packages the extension into a VSIX file. This file is used to install an extension for SSMS.

Add new features

  1. Right-click the project node in Solution Explorer and select Add>New Item.
  2. In the Add New Item dialog box, expand the Extensibility node under Visual C#.
  3. Choose from the available item templates: Editor Items (Classifier, Margin, Text Adornment, Viewport Adornment) and then click Add.

Don't try to add a Package, Command, Tool Window or Toolbox Control as SSMS does not support this right now.

The files for the template that you selected are added to the project. You can start adding functionality to your item template, press F5 to run the project, or add additional item templates.

Run and debug

To run the project, press F5. Visual Studio will:

  • Build the extension from the VSIX project.
  • Create a VSIX package from the VSIX project.
  • Run the VSIX installer.

You will need to increment the version number every time you build or the VSIX installer will not install the extension.
